phidgets Module

Defined in: lib/phidgets.js:2696

The Phidgets library module makes it easy to connect to various sensor and controller boards made by Phidgets Inc. This library works under Node.js and other compatible frameworks such as io.js

For support, go to the GitHub project page.

Please note that this library is an open source project that is not affiliated with Phidgets Inc.

This module provides the following classes:

  • Phidget
  • PhidgetBridge
  • PhidgetInterfaceKit
  • PhidgetLED
  • PhidgetRFID
  • PhidgetStepper
  • PhidgetTemperatureSensor
